# tirl-validator

Utilities to enable verifying TrustlessIRL physical labels. Built for React Native

## Installation

```sh
npm install tirl-validator
```

## Usage

```js
import { multiply } from 'tirl-validator';

// ...

const result = await multiply(3, 7);
```
